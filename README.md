# MAAL-Marcels-Amazing-Assembly-Language
This tool will compile MAAL into MAALB (Bytecode)

The Bytecode is planned to be run by MaslOS and possibly a windows interpreter.


## Example
```
loc MAIN:
  print "Hello, world!";
  exit;
```
